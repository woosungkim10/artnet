# 아트넷 (ArtNet) - AI 기반 시각예술 네트워크 플랫폼

## 프로젝트 개요

**김우성_시각예술 고립 해소를 위한 프로젝트**

시각예술가들의 인적 네트워크와 전시·공모 참여 기회의 접근 장벽을 제거하기 위한 AI 기반 플랫폼입니다.

## 문제정의

시각예술가들은 다음과 같은 어려움을 겪고 있습니다:

- 📍 **정보 접근 장벽**: 인적 네트워크와 전시·공모 정보의 접근이 제한적
- 🔗 **네트워크 고립**: 맥락적 연결과 담론의 흐름 파악 부족
- ⏰ **기회 상실**: 개인의 정보력에만 의존하여 기회 놓치기 쉬움
- 💡 **작가상 정립의 어려움**: 자신의 위치와 맥락을 파악하는 데 오래 걸림

## 솔루션

### AI 에이전트 군단 (G-Stack) 기반 접근

1. **파편화된 정보 분석**
   - 전시 카탈로그, 공모 공고 등 텍스트 자동 분석
   - AI가 핵심 정보와 맥락 추출

2. **맥락 시각화**
   - 작가-큐레이터-기관 간의 관계 자동 파악
   - 인터랙티브 네트워크 그래프로 시각화

3. **오케스트레이션 기법**
   - 여러 에이전트를 병렬적으로 수행
   - 효율적이고 빠른 분석

4. **개인 맞춤형 추천**
   - 당신의 작업 맥락에 맞는 연결과 기회 제공
   - 데이터 기반의 투명한 시스템

## 핵심 기능

- 📊 **전시·공모 분석**: AI가 자동으로 주요 정보 추출
- 🔍 **맥락 추출**: 인물 관계와 담론의 흐름 파악
- 🎯 **맞춤형 추천**: 당신의 스타일에 맞는 기회 추천
- 🌐 **네트워크 가시화**: 작가-큐레이터-기관 간 관계 시각화
- 💬 **커뮤니티**: 같은 맥락의 작가들과 연결
- 📈 **성장 추적**: 네트워크 성장과 기회를 데이터로 추적

## 기술 스택
